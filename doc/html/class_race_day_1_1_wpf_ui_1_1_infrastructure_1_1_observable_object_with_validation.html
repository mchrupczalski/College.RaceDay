bool ForceInitialErrorState [get, set]
 This property is used to force the initial error state of the object, before any property has been set. This is useful for when you want to show the error state of the object before the user has interacted with it. For example, when a dialog is opened, you may want to force the error state to set the CanExecute state of a command.
